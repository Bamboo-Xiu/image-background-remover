# Image Background Remover - MVP 需求文档

> 版本：v0.1 · 日期：2026-03-18 · 作者：修之竹

---

## 一、产品概述

### 1.1 产品定位
一个极简的在线抠图工具，用户上传图片，一键去除背景，直接下载结果。无需注册，无需登录，图片不存储。

### 1.2 目标用户
- 电商卖家（商品主图处理）
- 设计师（快速抠图）
- 普通用户（证件照换底色、社交媒体素材）

### 1.3 核心价值
- 快：上传即处理，秒级出结果
- 简：一个页面，三步完成
- 安全：图片不落盘，处理完即丢弃

---

## 二、技术架构

### 2.1 技术选型

| 层级 | 技术方案 | 说明 |
|------|----------|------|
| 前端 | 纯静态 HTML + JS + Tailwind CSS | 部署到 Cloudflare Pages |
| 后端 | Cloudflare Workers | API Key 保护层，转发请求 |
| AI 服务 | Remove.bg API | 商用级抠图效果 |
| 存储 | 无 | 图片仅在内存中流转 |

### 2.2 数据流
用户上传图片 → Cloudflare Pages（前端）→ Cloudflare Workers（代理）→ Remove.bg API → 返回结果 → 浏览器内存展示 → 用户下载

### 2.3 为什么选 Cloudflare Workers 做代理
- API Key 保存在 Workers 环境变量，不暴露到前端
- 免费额度 10 万次/天，足够 MVP 阶段
- 全球边缘节点，延迟低
- 与 Cloudflare Pages 原生集成，部署一体化

## 三、功能需求

### 3.1 MVP 核心功能（必做）

**F1 - 图片上传**
- 支持拖拽上传
- 支持点击选择文件
- 支持格式：JPG、PNG、WebP
- 文件大小限制：≤ 10MB（Remove.bg 免费版限制）
- 上传前预览原图

**F2 - 一键去背景**
- 点击「去背景」按钮触发处理
- 处理中显示 Loading 状态
- 失败时显示错误信息（如配额不足、格式不支持）

**F3 - 结果展示**
- 并排展示：原图 vs 去背景后
- 支持棋盘格背景（透明区域可视化）
- 支持切换纯白 / 纯黑 / 棋盘格预览背景

**F4 - 下载**
- 下载 PNG（保留透明度）
- 文件名：removed_bg_原文件名.png

### 3.2 MVP 不做（后续迭代）
- ❌ 用户账号系统
- ❌ 历史记录
- ❌ 批量处理
- ❌ 自定义背景替换
- ❌ API 接口对外开放
- ❌ 付费/积分系统

## 四、页面设计

### 4.1 页面结构（单页）

```
[Header]  Logo + 简短描述

[上传区]  拖拽区域，支持点击
          "拖拽图片到这里，或点击上传"
          支持 JPG / PNG / WebP，最大 10MB

[处理区]  左：原图预览
          右：结果预览（处理前灰显/骨架屏）
          中间：「去背景」按钮

[下载区]  「下载 PNG」按钮（处理完成后显示）

[Footer]  Powered by Remove.bg · 隐私说明
```

### 4.2 交互状态

| 状态 | 表现 |
|------|------|
| 初始 | 只显示上传区 |
| 已选图 | 显示原图预览 + 「去背景」按钮激活 |
| 处理中 | 按钮变为 Loading，右侧骨架屏 |
| 完成 | 显示结果图 + 「下载 PNG」按钮 |
| 失败 | 显示错误提示，可重试 |

## 五、API 设计

### 5.1 Workers 接口

```
POST /api/remove-bg

Request:
  Content-Type: multipart/form-data
  Body: { image_file: File }

Response（成功）:
  Content-Type: image/png
  Body: 去背景后的图片二进制流

Response（失败）:
  { "error": "错误信息", "code": "QUOTA_EXCEEDED | INVALID_FORMAT | API_ERROR" }
```

### 5.2 Remove.bg API 调用
- Endpoint: `https://api.remove.bg/v1.0/removebg`
- 认证：`X-Api-Key: ${API_KEY}`（存于 Workers 环境变量）
- 返回格式：image/png
- 免费额度：50次/月（正式上线需付费或按量计费）

## 六、部署方案

### 6.1 目录结构

```
/
├── public/
│   ├── index.html       # 单页前端
│   ├── style.css        # 样式
│   └── app.js           # 交互逻辑
├── workers/
│   └── remove-bg.js     # Cloudflare Workers 代理
├── wrangler.toml        # Workers 配置
└── README.md
```

### 6.2 部署步骤
1. 注册 Cloudflare 账号，获取 Remove.bg API Key
2. `npm install -g wrangler`
3. `wrangler secret put REMOVE_BG_API_KEY` 写入密钥
4. `wrangler deploy` 部署 Workers
5. Cloudflare Pages 连接 GitHub 仓库，自动部署前端
6. Pages 设置：Functions 路由 `/api/*` 指向 Workers

### 6.3 费用预估（MVP 阶段）

| 服务 | 费用 |
|------|------|
| Cloudflare Pages | 免费 |
| Cloudflare Workers | 免费（10万次/天） |
| Remove.bg | 免费 50次/月，超出按量计费 |

## 七、验收标准
- 用户可以上传 JPG/PNG/WebP 图片（≤10MB）
- 点击去背景后，30秒内返回结果
- 结果图背景透明，主体完整
- 可以下载 PNG 格式结果图
- API Key 不暴露在前端代码或网络请求中
- 页面在移动端可正常使用
- 处理失败时有友好提示
